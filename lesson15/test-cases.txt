5. Wait when element -> //h2/span[contains(text(), 'зубна щітка')] become visible
6. Find all elements //li[contains(@class, 'catalog-grid__cell')] and verify that amount > 0

Перевірити що можна додати товар в корзину із категорії “Зоотовари” -> “Корми для кішок”

2. Find and click on element -> //a[text()='Зоотовари']

4. Find and click element -> //li[@class='tile-cats__item']/a[@href='https://rozetka.com.ua/ua/food_for_cats/c1461212/'].
5. Wait when element -> //h1[contains(text(),'Корм для котів')] becomes visible.
6. Find element //div[@class='item'][2]//a[@class='tile-image-host'] and save its attribute ‘href’.
7. Find and click on element -> //div[@class='item'][2]//button[@class='buy-button'].
8. Wait when element -> //div[@class='item'][2]//button[contains(@class, 'buy-button_state_in-cart')] becomes visible and active and click on it.
9. Wait for element -> //rz-modal-layout becomes visible.
10. Find all elements -> //div[@class='cart-product'] and verify that there is only one such element
11. Get saved href and verify that element  //div[@class='cart-product__main’]//a[href=“saved_value”] is present.
